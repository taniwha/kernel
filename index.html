Just some quick links.

djgpp stuff

ed My own editor for djgpp. The tar file is uncompressed because it's really a collection of .zip files. Requires python and zlib. Vaguely (?:) similar to Borland's ide, but (IMHO) better, even though it's not finished yet (doesn't support spawning yet (eg running gcc), and the python integration is currently lowsy). Nice features include the ability to read gziped files and file size is effectively unlimited (reads linux-2.0.33.tar.gz without any problems, memory permitting).

orb 3D N-body gravitational orbit simulator. Uses allegro (sorry, can't find the url at the moment, but there's a link in the djgpp pages).

serio The latest version of my serial port code. An older version can be had from SIMTEL

M$ DOG LFN driver My (very unfinished) LFN driver (alpha, 0.1.3). Needs a much hacked version of DJ Delorie's djasm.

kernel My djgpp built os-kernel. Doesn't actually do anything yet, but can be uses as a starting point. Requires a working C and C++ capable djgpp, GNU bison, and djasm.
